# EmailToolAdviser Network — Weekly SEO Checklist

Run every Monday morning before noon ET. Budget: 30–45 minutes.

## 1. Search Console impressions check — all 5 domains
Pull the last 7 days of impressions for each property in [Google Search Console](https://search.google.com/search-console):
- emailtooladviser.com
- bestemailtoolreviews.com
- emailmarketingrated.com
- emailtoolratings.com
- smallbizemailhub.com

Compare to the previous 7 days. Flag any domain with a >20% week-over-week drop. (A drop on a single satellite is usually a content-pipeline issue; a drop across all 5 is usually a global Google update.)

## 2. Keywords that entered the top 10 this week
For each domain, sort the GSC "Queries" report by position descending, filter to position ≤10. Note any keyword that wasn't in the top-10 last week. These are the "rising stars" — they get an extra internal link from a top page and a copy refresh next week.

## 3. Articles that gained the most clicks
GSC → Pages → sort by clicks (delta vs previous 7 days). For each of the top 3 gainers:
- Confirm the page still has a strong CTA above the fold
- Confirm internal links to other articles are working
- Add one new internal link from a related article *to* this page

## 4. Content pipeline publish log
Open `automation/publish-log.json` (or check the GitHub Actions runs for `Daily Content Generation`):
- Confirm at least 3 articles published this week per active domain
- Note any failed runs — investigate the run log
- If the queue is below 5 days of capacity, append new keywords to `automation/keyword-queue.json`

## 5. Top 3 page-2 keywords to prioritize next week
GSC → Queries → filter to position 11–20 (page 2). Sort by impressions descending. Pick the top 3 that aren't already targeted by an existing article. Add them to `automation/keyword-queue.json` with priority 1 so they ship in the next pipeline run.

## 6. Technical errors in Search Console
For each property, check **Indexing → Pages**:
- Flag any non-zero "Not indexed" reasons
- Flag any new "Crawled — currently not indexed" pages (often a content-thinness signal)
Also check **Indexing → Sitemaps**:
- Confirm `sitemap.xml` was last fetched within the past 7 days
- Confirm "Discovered URLs" is ≥ "Submitted URLs"

## 7. Constant Contact affiliate dashboard — CPA check
Log in to the Constant Contact affiliate dashboard ([join.constantcontact.com](https://join.constantcontact.com/join-now) → affiliate partner login). Pull this week's conversions and EPC. Note:
- Total paid activations this week
- Top-referring URL across the network
- Any chargebacks or reversed conversions

## 8. Revenue calculation
This is the headline number for the week.

```
weekly_revenue = paid_activations × $650
month_to_date_revenue = (running total this calendar month)
projected_monthly_revenue = (weekly_revenue ÷ days_elapsed) × 30
```

Log the result to `data/revenue.json` so the dashboard at `/dashboard/` picks it up.

## When something is broken

- **Pipeline failed:** check `https://github.com/ecombrandsai/emailtooladviser/actions/workflows/content-pipeline.yml`
- **Deploy failed:** check the same Actions tab, `deploy.yml` workflow
- **GSC missing data:** verify `automation/gsc-credentials.json` is fresh in CI secrets
- **Dashboard empty:** verify `/data/*.json` files have been updated this week
